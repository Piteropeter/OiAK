# OiAK

To build OiAK library just run 

    prepare_sdk.sh

and after all dependencies are ready `cd` into `build/` catalog and type

    make

And that's all! Unit test are localized in `ut/unit_tests` and benchmarks in `benchmark/benchmarks`

Enjoy!
